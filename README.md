# Claude Skills (Web)

Коллекция навыков (skills) для [Claude.ai](https://claude.ai) — веб-версии, работающие с текстом без серверного окружения.

## Установка

### Способ 1: Скачать ZIP из Releases

1. Перейди в [Releases](https://github.com/bibliomantus/claude-skills-web/releases)
2. Скачай ZIP-файл нужного навыка
3. Зайди в claude.ai → **Customize** → **Skills**
4. Нажми **"+"** → **"Upload a skill"**
5. Загрузи ZIP-файл

### Способ 2: Скачать вручную

1. Скачай папку навыка
2. Запакуй её в ZIP (важно: в архиве должна быть **папка**, а не просто файлы)
3. Загрузи через Customize → Skills → Upload

## Навыки

### action-extractor

**Разбор материалов в пошаговые руководства**

Анализирует текст (транскрипт видео, статью, книгу, лекцию) и создаёт исчерпывающее практическое руководство с пошаговым планом действий на русском языке.

**Как использовать:** вставь текст материала в чат и скажи «разбери материал», «составь пошаговый план» или «extract actionable steps».

Что делает:
- Сначала показывает **предварительный обзор** — тема, тип содержимого, целевая аудитория
- Извлекает конкретные действия из хаотичного материала
- Приоритизирует шаги: быстрые результаты → основные навыки → продвинутый уровень
- Определяет пререквизиты — что нужно изучить до выполнения шагов
- Добавляет **ловушки новичка** — типичные ошибки с привязкой к конкретным шагам
- Добавляет **адаптацию под контекст** — варианты для фрилансера / команды / ограниченного бюджета
- Собирает **таблицу ресурсов** — все инструменты и сервисы из материала
- Генерирует **компактный чеклист** для печати
- Поддерживает многочастные материалы (серии видео/статей)

Триггеры: «разбери материал», «проанализируй», «составь пошаговый план», «сделай разбор», «extract actionable steps», «break down this text», «turn into a guide»

> Полная версия с автоматическим скачиванием YouTube-транскриптов и статей доступна для [Claude Code](https://docs.anthropic.com/en/docs/claude-code).
